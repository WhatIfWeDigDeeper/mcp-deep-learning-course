# MCP Deep Learning Course

Notes and code for the DeepLearning.AI course [MCP: Build Rich-Context AI Apps with Anthropic](https://learn.deeplearning.ai/courses/mcp-build-rich-context-ai-apps-with-anthropic/).

## Project Setup

- Python project managed with `uv`
- Jupyter notebooks in `src/`
- Requires `.env` file with `ANTHROPIC_API_KEY`

## Dependencies

- `anthropic` - Claude API client
- `arxiv` - arXiv paper search
- `python-dotenv` - Environment variable loading
- `ipykernel` - Jupyter kernel support
